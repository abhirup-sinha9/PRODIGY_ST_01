# 🧮 Calculator Testing Project

## 📌 Overview
This repository contains structured manual test cases for validating a basic calculator application.  
It focuses on verifying arithmetic operations, BODMAS rules, decimal handling, and error validation.

---

## ✨ Features Covered
- Addition, Subtraction, Multiplication, Division  
- Positive and negative numbers  
- Decimal calculations  
- BODMAS / Order of Operations  
- Division by zero handling  
- Invalid input validation  

---

## 🧪 Test Cases
All functional test cases are documented in:

📄 **TEST_CASES.md**

The test cases include:
- Valid input scenarios  
- Invalid input scenarios  
- Preconditions and test steps  
- Expected results  

---

## ▶️ How to Use
1. Open the calculator application.  
2. Refer to `TEST_CASES.md`.  
3. Execute test steps as mentioned.  
4. Compare actual output with expected results.  
5. Mark test cases as Pass or Fail.
